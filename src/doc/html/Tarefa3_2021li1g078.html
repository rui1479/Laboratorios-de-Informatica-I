tudo :: Jogo -> String #

A função Tudo é a função que transforma os dados Jogo em Strings podendo assim ver/ler o jogo.

Exemplos de utilização:

>>> tudo (Jogo [[Vazio,Vazio],[Bloco,Bloco]] (Jogador (1,0) Oeste False))
" <\nXX"

Propriedades:

"/n" reprensenta a troca de linha.

" " representam os Vazios.

C representam as Caixas.

X representam os Blocos.

">" ou"<" representam o jogador e a sua respetiva direção.

O jogador também pode ter uma caixa, e nesse caso a caixa aparece por cima deste.

O jogador so pode pegar numa caixa se esta tiver a frente do jogador.
